1) Get access to your Subscription/AKS instance. 
Go on your terminal or cloud shell and enter these commands:

az login --tenant 16b3c013-d300-468d-ac64-7eda0820b6d3
az aks get-credentials -g MSUS_SA_SupplyChain -n sa-aks-f8v2d19903987e --overwrite-existing

2) Instantiate a KubeRay Operator on your aks:
kubectl create -k "github.com/ray-project/kuberay/ray-operator/config/default?ref=v0.3.0&timeout=90s"

3) From the local folder where you saved you raycluster autoscaler yaml file, run this command:
kubectl apply -f ray-cluster.autoscaler.yaml

4) Open the ports for the Ray Dashboard in a different terminal (it's a blocking command):
kubectl port-forward service/raycluster-austoscaler-head-svc 8265:8265

--------------- Other useful commands below

To delete the RayCluster on your terminal, run this command:
kubectl delete raycluster raycluster-autoscaler

To delete the RayOperator in the ray-system namespace, run this command: 
kubectl delete -k "github.com/ray-project/kuberay/ray-operator/config/default?ref=v0.3.0&timeout=90s"

Some useful kubectl commands:

kubectl get pods -A
kubectl get service raycluster-autoscaler-head-svc
kubectl describe pod supplychainray220-577f4bd7c9-rd2qg -n azureml-mlw-sa-httv3
kubectl exec -it raycluster-autoscaler-worker-small-group-tqdgp sh
